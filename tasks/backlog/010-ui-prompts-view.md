## Description
Build the prompts tab UI. Shows a list of prompts ordered by date. Clicking a prompt opens a detail view with the current content rendered as markdown, an edit mode, and a revision history panel showing all past revisions with timestamps and diffs or change summaries.

## Acceptance Criteria
- [ ] Prompts list view with title, date, revision count
- [ ] Create new prompt flow
- [ ] Prompt detail view with rendered markdown
- [ ] Edit mode that saves and creates a new revision
- [ ] Revision history panel listing all revisions
- [ ] Ability to view any past revision
